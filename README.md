## LumiChat
LumiChat is a real-time chat application designed to provide secure and seamless communication between users. The application allows users to sign up, log in, and send messages including text and files, with support for a user-friendly interface. Built with a focus on simplicity, speed, and security, LumiChat utilizes modern web technologies to ensure a smooth experience.

# Features
- User Authentication: Users can sign up and log in to the chat system securely.

- Real-Time Messaging: Send and receive text messages in real-time.

- File Uploads: Attach and send files seamlessly during chats.

- Responsive Design: Fully responsive UI for a pleasant experience across all devices.

- Secure Backend: All communication and data are securely handled through a robust API.

# Tech Stack
- Frontend: HTML, CSS, JavaScript

- Backend: Node.js, Express, Azure Functions

- Database: Azure Cosmos DB 

- File Storage: Azure Blob Storage

- Hosting: Azure Static Web Apps for frontend and Azure Functions for the backend API

# Getting Started
1. Clone the Repository
```bash
  git clone https://github.com/Vedashree14/lumichat.git 
  cd lumichat
```
2. Install Dependencies
   
- Run the following command in both the frontend and backend directories:
```bash
npm install
```

3. Running the Application Locally
   
- Start the frontend (in the frontend directory):
```bash
npm start
```
- Start the backend (in the backend directory):

```bash
npm start
```
The application will be available at http://localhost:3000 for the frontend, and the backend API will run at http://localhost:7071/api.
